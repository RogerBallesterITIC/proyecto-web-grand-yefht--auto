# 🌆 Proyecto web Grand Yheft Ñauto
Este proyecto es una propuesta de rediseño creativo que traslada el lenguaje visual de la legendaria saga Grand Theft Auto al entorno del desarrollo web moderno. No se trata solo de una copia estética, sino de una adaptación funcional que toma elementos icónicos

---
# 🛠️ Características Principales

* **Diseño Inmersivo:** Uso de paletas de colores neón y tipografía personalizada (Pricedown style).
* **Interfaz Dinámica:** Menús interactivos que emulan la navegación dentro de Grand Yheft Ñauto.
* **Estética Cinematográfica:** Fondos de alto contraste hechos con IA.
* **Responsividad:** Adaptado para que la experiencia se mantenga en diferentes tamaños de pantalla.
---
## 💻 Tecnologías Utilizadas

* **HTML5 / CSS3:** Estructura y diseño visual.
* **Google Fonts:** Tipografía para legibilidad.
